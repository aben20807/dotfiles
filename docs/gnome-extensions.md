# Gnome Extensions

## Install

+ Link: [Gnome Extensions (use firefox to open)](https://extensions.gnome.org/)
+ Install extension in firefox

```bash
$ sudo apt install -y gnome-shell-extension-manager # old: chrome-gnome-shell
```
